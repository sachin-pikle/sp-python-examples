# Pygal Example

https://www.pygal.org/  
https://pypi.org/project/pygal/

```
cd pygal

pyenv versions

pyenv shell 3.14.2

python --version

python -m venv .venv

source .venv/bin/activate

which python

pip install --upgrade pip

pip install pygal
```

**Write code. Run it using:** 

```
python example_pygal.py
```

**Deactivate**

```
deactivate
```
